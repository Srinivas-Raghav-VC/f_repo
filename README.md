*Machine unlearning in multilingual LLMs is harder than in monolingual
settings: a single concept (e.g., Hindi) spans multiple scripts and leaks through related
languages. We present MMIE, a research pipeline that pairs sparse autoencoders (SAEs)
with lightweight adapters (LoRA/ReFT) and a seven-gate, FDR-corrected evaluation to
test whether a low-resource language can be cleanly suppressed without collateral
damage.*
